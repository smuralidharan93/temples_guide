# Song Represented Shivan Temples Guide

A travel and tracking guide for the 276 Paadal Petra Sthalam (Song Represented Shivan Temples).

This application is built as a **Progressive Web App (PWA)**, meaning you can install it directly to your home screen without going through the App Store or Play Store.

## Step 1: Open the App
Open the `index.html` file in your mobile browser.
*(Note: To install as PWA, you might need to host this on a static web server like GitHub Pages or Netlify. If opening locally, it works as a webpage)*.

## Step 2: Install on Your Device

### Android (Chrome)
1. Tap the **Menu** (three dots) in Chrome.
2. Select **"Install App"** or **"Add to Home Screen"**.
3. Confirm by tapping **Add**.

### iOS (Safari)
1. Tap the **Share** button (box with arrow up) at the bottom.
2. Scroll down and tap **"Add to Home Screen"**.
3. Tap **Add** in the top-right corner.

The app will now appear on your home screen with the app icon and launch in full-screen mode like a native app.

## Features

- **276 Temples**: Complete directory with details.
- **Offline Mode**: Works without internet once loaded.
- **Visisted / Bucket List**: Track your pilgrimage.
- **Maps**: Direct navigation links.
